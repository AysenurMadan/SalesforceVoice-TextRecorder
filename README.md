Kurulum sadece 30 saniye sürer ve tamamen ücretsizdir.

1. Adım: Size gönderdiğim .zip dosyasını masaüstüne indirin ve içindeki klasörü dışarı çıkartın.

2. Adım: Google Chrome'u açın ve adres çubuğuna şu yazıyı yapıştırıp Enter'a basın: chrome://extensions

3. Adım: Açılan sayfanın sağ üst köşesindeki "Geliştirici Modu" (Developer Mode) anahtarını açın.

4. Adım: Masaüstüne çıkardığınız klasörü farenizle tutup bu sayfanın ortasına sürükleyip bırakın.

Eklenti yüklenecektir! Sağ üstteki "Puzzle" simgesine tıklayıp Kitoko Voice'u sabitleyebilirsiniz.

Not: Kullanırken mikrofon izni isterse "İzin Ver" demeyi unutmayın.
Not: Klasörü bilgisayarınızdan silmeyin (silinirse eklenti gider)
